OPTIMAQ – Resource Load Optimizer (Java Implementation)

Author:
Tasmir Khan

Project Description:

OPTIMAQ is a resource allocation and task scheduling system that distributes tasks among available resources efficiently using greedy optimization techniques. The goal is to minimize overload, improve utilization, and detect bottlenecks.

Core Features:

• Priority based scheduling
• Greedy resource allocation
• Load balancing
• Resource utilization tracking
• Bottleneck detection
• Resource efficiency ranking
• Task rejection analysis

How To Run:

1 Go to src folder

2 Compile:

javac main/Main.java models/*.java optimizer/*.java

3 Run:

java main.Main

Algorithm Used:

Greedy Scheduling Algorithm:
Tasks are sorted by priority and allocated to the least loaded resource that can handle the task.

Objective:

To demonstrate algorithmic optimization techniques for efficient resource management.
