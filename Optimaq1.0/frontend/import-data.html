Data Format Guide

Supported Field Names

Category Recognized Field Names
ID/Name id, name, resourceName
Capacity capacity, max_capacity, totalBeds, total_capacity
Current Load load, current_load, occupiedBeds, occupied
Tasks/Activities tasks, activities, allocations (optional)

Example JSON Format

[
  {
    "id": "ICU_Ward",
    "capacity": 20,
    "load": 18,
    "tasks": ["Patient_Care", "Monitoring"]
  },
  {
    "id": "ER_Ward",
    "capacity": 15,
    "load": 12,
    "tasks": ["Emergency_Response"]
  }
]

Example CSV Format

id,capacity,load,tasks
ICU_Ward,20,18,"Patient_Care,Monitoring"
ER_Ward,15,12,"Emergency_Response"
