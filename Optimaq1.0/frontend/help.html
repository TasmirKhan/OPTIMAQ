Help Center

Get started quickly with OPTIMAQ

Quick Start

  1. Go to Import and upload a JSON/CSV file.
  2. Open Dashboard and click Run Optimization.
  3. Review insights in Analytics and Resources.

Troubleshooting

  • If data is missing, re-import from the Import page.
  • If redirected, login with admin / optimaq123.
  • Use Refresh in Dashboard after importing data.
